Gates library

The Gates library includes a variety of simple components, all of which have a single output whose value is dictated entirely by the component's current inputs.

NOT Gate — inverts a signal
Buffer — does nothing, no-op
AND, OR, NAND, NOR Gates — simple logic operations
XOR, XNOR, Odd Parity, Even Parity — equality and parity operations
Tri-state Buffer, Tri-state Inverter — gates with three possible outputs
PLA — Programmable Logic Array
