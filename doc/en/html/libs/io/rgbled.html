RGBLED

Library: Input/Output
Introduced: 3.x.x
Appearance:

Behavior

Displays the value of its three inputs as one of eight possible colors.

Pins

A RGBLED has three pins, one representing red, one for green, and one for blue. When all inputs are 0, the LED appears black. WHen all are 1, the LED appears white. Otherwise the LED appears colored according to the RGB value of the three inputs.

Attributes

When the component is selected or being added, the arrow keys alter its Facing attribute.

Facing
The location of the input pin relative to the component.
Active On High?
If yes, then each RGBLED input is considered "On" when the input is 1, and "Off" otherwise. If no, the input is "On" when the input is 0, and "Off" otherwise.
Label
The text within the label associated with the component.
Label Location
The location of the label relative to the component.
Label Font
The font with which to render the label.
Label Color
The color with which to draw the label.

Poke Tool Behavior

None.

FPGA Synthesis

Supports VHDL and Verilog synthesis, as if it were three separate LEDs (one for R, one for G, and one for B).
