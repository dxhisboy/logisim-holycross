DIP Switch

Library: Input/Output
Introduced: 3.x.x
Appearance:

Behavior

Outputs a set of zeros or one, as defined by the user-controlled switches.

Pins

A DIP Switch has as multiple output pins, as determined by the 'Number of Switch' attribute.

Attributes

When the component is selected or being added, Alt-0 through Alt-9 alter its Number of Switch attribute, and the arrow keys alter its Facing attribute,

Facing
The location of the output pin relative to the component.
Label
The text within the label associated with the component.
Label Location
The location of the label relative to the component.
Label Font
The font with which to render the label.
Label Color
The color with which to draw the label.
Number of Switch
How many switches and outputs the component has.

Poke Tool Behavior

Pressing the mouse button will toggle one of the switches from 0 to 1 or vice versa.

FPGA Synthesis

Supports VHDL and Verilog synthesis.
