Analog library

The Analog library includes components that normally would not be necessary for a purely digital design.

Warning: Logisim's circuit simulator is a discrete event simulator using a four-valued logic: each bit can be zero (0), one (1), error (E), unknown (X). Propagation of signals through gates and wires is simulated in a very crude fashion, usually just one time step per gate or component and instantaneous transmission along wires. As such, only a very crude approximation is made to simulate the analog behavior of components. The components in this library are simplistic, and only meant for the most basic of educational purposes.
Power, Ground — signal source and sink
Transistor — simplified PNP and NPN transistors
Transmission Gate — block or transmit a signal
Pull Resistor — weakly pull signal to a default value
