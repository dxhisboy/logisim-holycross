Mouse Tool Library

The Mouse Tool Library includes tools for editing and interacting with circuits. This library is only visible when you right-click the toolbar and select "Customize...", as these tools normally only appear in the toolbar.

Poke Tool — interact with running simulation
Multi-Tool — select and move things, or draw wires
Move Tool— just select and move things
Wiring Tool — just draw wires
Wire Cutter — delete wires or parts of wires
Text Tool — add or edit labels or text comments
Menu Tool — bring up the right-click context menu
Comment — add text comments
